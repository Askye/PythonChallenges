# Trigonométrie - Calculer la mesure principale

## Notions abordées

- Boucle et Condition
- La fonction abs()

## Énoncé

Vous devez calculer la mesure principale d'un angle.

On entre la valeur de l'angle sous la forme N/D*pi :

```
Numérateur   : -246
Dénominateur : 5
```

Python devra afficher :
```
La mesure principale est 4/5 π
```
