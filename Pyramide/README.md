# Pyramide

## Notions abordées

- Boucle et Fonction

## Énoncé

Vous devez créer une fonction qui prend une hauteur en argument et qui affiche une pyramide en sortie.

Exemple :
```
hauteur = 10
```

Résultat :
```
         x                         1
        xxx                        3
       xxxxx                       5
      xxxxxxx                      7
     xxxxxxxxx                     9
    xxxxxxxxxxx                    11
   xxxxxxxxxxxxx                   13
  xxxxxxxxxxxxxxx                  15
 xxxxxxxxxxxxxxxxx                 17
xxxxxxxxxxxxxxxxxxx                19
```
